# Proyecto Arquitectura de Ordenadores 2024

## Como hacer el build.
### Requisitos: 
- Tener instalado `cmake`: `sudo apt install cmake`
### Build 
Desde una terminal en el directorio donde tengais el proyecto:
```bash 
mkdir build
cd build
cmake ..
make --build .
```
